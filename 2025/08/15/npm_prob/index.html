# nodejs與hexo安裝問題

在安裝完nodejs軟體後要先建立路徑以使指令可被找到

  1. 先去系統中的環境變數設定

  2. 在本機設定的path中新增C:\Program Files\nodejs\

  3. 儲存後重開機並在cmd中輸入npm -v 要顯示出你安裝的版本號

  4. 安裝hexo則在終端機中輸入npm install hexo-cli -g 以安裝hexo全域指令與工具

  5. 在使用者設定的path中新增C:\Users\newadmin\AppData\Roaming\npm

  6. 儲存後重開機並在cmd中輸入hexo -v 要顯示出你安裝的套件版本

# 問題1:powershell找不到npm指令

進行1-3步驟後以windows11來說發現cmd可以使用npm但powershell找不到

-> 發現是ps1檔案權限遮蔽的問題

  • 原因:

-> 因C:\Program Files\nodejs中包含npm.cmd,npm.ps1等檔案

而ps1是powershell的設定檔案且因這種檔案會被預設擋住(Execution Policy)無法執行而直接報錯

但這種檔案不是給cmd的(cmd讀不到)所以她不受影響

  • 解法:

    1. 先以管理員權限開啟power shell 輸入Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned(可自動執行$profile設定黨)

    2. notepad $PROFILE 新增一個額外的系統設定檔

    3. 內輸入Set-Alias npm "C:\Program Files\nodejs\npm.cmd" Set-Alias npx "C:\Program Files\nodejs\npx.cmd" 儲存

    4. 關掉所有舊視窗重開測試npm -v是否有成功運作
