RHCSA200 考試心得

RHCSA200 考試準備流程與通過心得

2025-08-15

考試動機 #

在參加學校選修的課程有一定的基礎後決定順便考取這個證書

報名與考試流程 #

參加 RECSA200 考試的方式有兩種：

  1. 在家遠端考試 → 需自備有線滑鼠,攝像頭,usb
  2. 直接去他們的指定定點考試 → 台灣只有台北有(但就帶證件就好不需其他東西)

而考試卷的購買流程可以參考

考試卷購買流程(來自:想吃地瓜酥的藍熊)

在預約考試流程那邊選擇你想要遠端還是去指定地方就好了
下面就只介紹在家遠端考試的流程(指定地點其實就是聯成電腦,他們提供所有東西)

在家遠端考試 #

安裝考試環境流程

  1. 在確定考試後會收到email說明考試環境的安裝方式跟規則
  2. 跟著步驟,先下載考試用iso檔案,並按照影片選擇適用的方式將iso檔案燒入usb
  3. 依照每個電腦不同,切換到開機選單選擇用usb磁碟(有些電腦可能要先去關掉安全開機)
    使用windows11的人要小心(在切換安全開機後)要是你的使用者有綁微軟帳號,他可能會封鎖pin碼要求你登入重設密碼
  4. 在考試前可以先進入考試環境檢查環境設備是否符合要求
  5. 在考前10分鐘先進入考試跟考官做身分,環境檢查

P.S. 注意環境要求攝像頭,有線滑鼠

考試資訊與評分標準 #

  • 題型：多個電腦設定要求
  • 通過分數：70%（210/300）分
  • 時間限制：180分鐘
  • 內容範圍：官方證照資訊

考試準備方法 #

1. 直接上他們的官方教材RH124.RH199 #

他們有提供完整的教材教學跟模擬機環境提供你練習

2. 參考書籍與考古題 #

  • 我常看的linux教學頻道
  • 推薦書籍

基本上我是透過學校的教材及課堂練習,並練習了許多題目(不懂跟較少的觀念會找youtube跟網路文章學習)這張證照考驗你掌握linux功能的熟練度,建議是直接買教材學習才不會學太亂太雜,畢竟也有許多版本跟系統不同的差別

總結與心得 #

  • 熟悉 linux 的系統操作
  • 多做考古題，理解每題為什麼對或錯
  • 架個虛擬機模擬linux的操作來熟練他
